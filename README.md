# Consultation Recording Manager

## Project Overview

A simple web application to manage consultation recordings. Users can view a dashboard, upload consultation details, and see a list of recordings.

## Tech Stack

* React (Frontend)
* Node.js
* Express.js
* React Router

## Features

* Dashboard page
* Upload Recording page
* View Recordings page
* Basic backend APIs

## Assumptions

* Data is stored in memory instead of a database.
* Authentication and file uploads are not implemented due to the limited assignment timeframe.

## Future Improvements

* MongoDB integration
* User authentication (JWT)
* Audio file uploads
* Search and filtering
* Dashboard analytics

## How to Run

### Backend

npm install
npm run dev

### Frontend

npm install
npm run dev
